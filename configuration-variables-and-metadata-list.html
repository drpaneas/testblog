Configuration Variables

These are the optional configuration variables that you can define

  1. PROJECTS used for Projects feature on the landing page
  2. USE_SHORTCUT_ICONS used for shortcut icons like favicon, Opera Speed Dial
  3. SITE_LICENSE used to add site license
  4. SITE_SUBTITLE used to add site subtitle in the footer
  5. COMMENTS_INTRO used with comments
  6. SITE_DESCRIPTION used for Search Engine Optimization
  7. LANDING_PAGE_ABOUT used for About me feature on the landing page
  8. SOCIAL_PROFILE_LABEL used with social media profiles widget
  9. RECENT_ARTICLES_COUNT see the post Recent Articles

These are the variables used for Mailchimp subscriber form in the sidebar,

  1. MAILCHIMP_FORM_ACTION is mandatory. Set it to your Mailchimp form action URL
  2. SUBSCRIBE_BUTTON_TITLE
  3. EMAIL_FIELD_PLACEHOLDER
  4. EMAIL_SUBSCRIPTION_LABEL

These are the variables used with different Pelican plugins,

  1. SERIES_TITLE used with multi_part plugin
  2. SHARE_POST_INTRO used with share_post plugin
  3. RELATED_POSTS_LABEL used with related_posts plugin

These are the variables used for Social Media Optimization,

  1. FEATURED_IMAGE
  2. TWITTER_USERNAME
  3. GOOGLE_PLUS_PROFILE_URL

Blog onCrash=Reboot(); uses Elegant theme. You can see its configuration files at Github for inspiration.

Metadata

Pelican uses file metadata from your articles and pages text files to get information about your posts, like tags, authors etc.

On top of “official” metadata, Elegant uses some optional metadata that you can use in your articles and pages.

  1. summary used for Search Engine Optimization
  2. subtitle used to set article subtitle
  3. keywords used for Search Engine Optimization
  4. modified used to set last updated time of the article
  5. series_title used with multi_part plugin
  6. featured_image used for Social Media Optimization
  7. comments_intro used with comments
  8. share_post_intro used with share_post plugin
  9. disqus_identifier used with comments
